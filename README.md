# Esar-i-Ziya-Pasa-OCR-d
This repository includes the OCR'd output for Ziya Paşa's Eşar-ı Ziya.

The data is processed with the OCR model for printed Ottoman Turkish [Ottoman-Print-OCR-Qwen4B](https://huggingface.co/enesyila/Ottoman-Print-OCR-Qwen4B). The lines were detected with a custom YOLO26s model.

As approximately 30% of this book is part of the training data, the OCR output might be better than usual.
